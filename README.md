# API REST con Node.js y Express

**Este proyecto es una API REST sencilla desarrollada con Node.js y Express.Permite realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre una lista de usuarios almacenada en memoria local.**

# Instalación
- **Asegúrate de tener instalado:**
- **Node.js y npm (Necesario para ejecutar el código).**
- **Visual Studio Code (Editor recomendado).**
- **Thunder Client: Instala esta extensión dentro de VS Code para probar las rutas (POST, PUT, DELETE).**
